改进一：GRN 全局响应归一化

GRN(Xi) = γ · Xi / √(‖Xi‖² + ε) + β

🔬 增强多样性

沿通道维度 L2 归一化
各通道响应被重新标定

🛡️ 抑制坍缩

强制各通道保持
不同的响应模式

🎯 突出显著性

高激活区域被放大
背景噪声被抑制

ConvNeXt V2 = V1 + GRN (每个 block 末尾) + FCMAE 自监督预训练 (60% 掩码)

GRN的核心作用——在特征图每个空间位置沿通道做L2归一化。三个效果：增强多样性、抑制特征坍缩、突出重要区域。

改进二：GeM Pooling 广义均值池化

f(g) = [ 1/|Ω| · Σ xp ]1/p

p = 1

≡ 平均池化 (GAP)
所有位置等权重

p > 1

聚焦高响应区域
保留全局信息

p → ∞

≡ 最大池化 (GMP)
仅保留最强位置

⚡ 关键设计决策：pinit = 1.0 (非文献默认 3.0)
p=3 导致首轮 Val Acc 仅 5.9% → 随机特征的噪声被 GeM(p=3) 放大 → 收敛崩溃
p=1 从 GAP 起步 → 训练中 p 自动增长 → 首轮恢复到正常水平
GeM的核心：可学习的p参数在avg和max之间调节。关键创新：p_init不用文献默认的3.0，而是1.0——因为3.0会放大随机噪声导致训练崩溃。

改进三：EMA 指数滑动平均

θshadow(t) = 0.999·θshadow(t-1) + 0.001·θ(t)

📊 平滑波动

等效窗口 ~1000 步
降低单 batch 噪声影响

🔄 等价集成

Polyak 平均在线近似
零额外推理开销

🎯 改善小样本

小样本类别参数估计更稳定
Candidiasis (275例) F1 +4.2%

decay 选择：0.99 (窗口100步 → 不够) | 0.999 (窗口1000步 → 最优) | 0.9999 (窗口10000步 → 太慢)

EMA维护参数的滑动平均副本，推理时用shadow权重。decay选0.999是经验最优——平衡平滑度和响应速度。

消融分析：组件贡献拆解

GRN ≈ +1.2%  |  GeM ≈ +0.5%  |  EMA ≈ +0.7%  |  协同效应 > 简单相加

三个组件各自贡献约1.2%/0.5%/0.7%。协同效应大于简单相加——GRN增强特征多样性，GeM在多样化特征空间里更容易找到判别区域。
