改进一：GRN 全局响应归一化

GRN(Xi) = γ · Xi / √(‖Xi‖² + ε) + β

对每个空间位置沿通道维度做 L2 归一化，插入每个 ConvNeXt block 末尾

增强特征多样性

归一化后各通道响应被重新标定

抑制特征坍缩

强制各通道保持不同响应模式

突出显著区域

高激活区域被放大，背景被抑制

ConvNeXt V2 = V1 + GRN (每个 block) + FCMAE 自监督预训练

改进二：GeM Pooling 广义均值池化

f(g) = [ 1/|Ω| · Σ xp ]1/p

p = 1

退化为平均池化（GAP）
所有位置等权重

p = 3

聚焦高响应区域
保留全局信息

p → ∞

逼近最大池化（GMP）
仅保留最强位置

关键设计决策：pinit = 1.0 (非文献默认 3.0)

p=3 导致首轮 Val Acc 仅 5.9%（噪声放大） → p=1 从 GAP 起步，训练中自动增长

改进三：EMA 指数滑动平均

θshadow(t) = 0.999 · θshadow(t-1) + 0.001 · θ(t)

平滑参数波动

等效窗口 ~1000 步
降低单 batch 噪声

等价模型集成

Polyak 平均的在线近似
零推理开销

改善小样本

小样本类别参数估计更稳定
Candidiasis F1 +4.2%

decay 选 0.999：平衡平滑度 (0.99 不够) 与响应速度 (0.9999 太慢)
